# Sales Data Analysis Project

## Overview
End-to-end sales data analysis using Python (Pandas, NumPy, Matplotlib) and Power BI.  
Cleaned and analyzed 4,000+ sales records to generate actionable insights for business decisions.

### Key Insights
- Top-performing products identified  
- Monthly sales trends analyzed  
- Region-wise performance evaluated  
- Customer behavior studied (new vs repeat)  
- Clean CSV files exported for Power BI dashboard creation

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)  
- SQL (for querying data, if applicable)  
- Power BI (dashboards & visualizations)  
- Excel (additional data analysis)

## How to Use
1. Clone the repository:
2. Open Jupyter Notebook (if included) to follow the analysis step-by-step.  
3. Run Python scripts to preprocess and analyze data.  
4. Import exported CSV files into Power BI for dashboards.
